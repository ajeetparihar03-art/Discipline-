# Discipline-

## How to add index.html file

1. Create a new file named `index.html` in your project directory
2. Add the basic HTML structure:
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Discipline</title>
</head>
<body>
    <h1>Welcome to Discipline</h1>
</body>
</html>
```
3. Save the file
4. Open it in your web browser to test
